.. note:: :ref:`Building the Car <doc_build_car>`, :ref:`System Configuration <doc_software_setup>`와 :ref:`Installing Firmware <doc_build_car_firmware>`를 완료했다고 가정한다.

| Vehicle built? ✓ System configured? ✓ Firmware installed? ✓

	* 완성된 F1TENTH 차량
	* Pit/Host 컴퓨터

**난이도:** 중

**걸리는 시간:** 2 시간
